# pdn-photoshop-brush

A [Paint.NET](http://www.getpaint.net) FileType plugin that loads and saves Adobe® Photoshop®-compatible brushes.

## How to install the plugin

1. Close Paint.NET.
2. Place ABRFileType.dll in the Paint.NET FileTypes folder (usually C:\Program Files\Paint.NET\FileTypes).
3. Restart Paint.NET.
